# Contributing to Shipping Calculator

Thank you for your interest in helping us optimize our supply chain tools! We welcome contributions from all teams and community members.

## How to Contribute

### 1. Reporting Bugs
If you find a bug in the shipping calculation logic or UI, please open an Issue in this repository. Include:
* A clear description of the bug.
* Steps to reproduce the issue.
* Expected vs. actual behavior.

### 2. Suggesting Enhancements
Have an idea for a faster routing algorithm or a clearer delivery estimate view? Open an Issue with the tag `enhancement` to discuss it.

### 3. Submitting Code Changes
To contribute code fixes or features:
1. **Fork** this repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. Commit your changes with clear, descriptive commit messages.
4. Push your branch to your fork.
5. Open a **Pull Request** against our `main` branch.

## Code of Conduct
Please note that this project is released with a Contributor Code of Conduct. By participating in this project, you agree to abide by its terms.
